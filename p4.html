Tilt Game

Now we are going to use the knowledge we gained in Project 2 to create our very own interactive tilt video game.
  1. Start by loading up the project you made during Project 2. It should provide you with a custom sprite that moves around the screen when you tilt the Arduino.
  2. Now we need to add a custom background and a few more sprites that we will "collect" by tilting our character around the screen.

    My game will feature a starfish that will move around underwater collecting muffins.
  3. Let's begin by programming the starfish. Make sure you have the starfish selected.

  4. We will use the same code we used in Project 2, but this time let's make sure we can move the sprite along the x and y axis.

  5. At this point you should be able to move your sprite around the screen by clicking on the green flag and tilting the Arduino. However, there is still something we are missing. We need to be able to "collect" the other sprites (in my example the muffins), and we should keep score.

    Go to the Data tab and click the Make a Variable button. In programming, we use variables to store information. In this example, the variable will be storing our score. Let's name our variable "score".

  6. The last thing that we will need to program on our main character is to reset the score each time we start the game. Since we start the game by clicking the green flag, this will be the best place to set the score back to 0.

  7. Now we still need to program it so that the main character can "collect" the other sprites and grant us points. First, select the other sprite that we want to program. In my example it is the muffin.

  8. Let's program the muffin to wait a second before appearing, then when the starfish touches it we will add a point and hide the muffin. After a second the muffin will reappear and can be collected again.

  9. We can duplicate the muffin sprite (along with it's code) a few times so that we will have a few different sprites to pick up. Scatter them around the screen.

  10. Now see if you can extend the game.
    Can you:
    • Make the other sprites (muffins) reappear at different time intervals?
    • Add other sprites that are worth different point values? Maybe some that shouldn't be collected or they take away points?
    • Advanced: Use the to make the sprite reappear at random intervals?
    • Advanced: Use the to make the sprite reappear at random locations?
